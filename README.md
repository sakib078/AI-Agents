# Agentic-AI

Notes and projects from the 🤗 [Hugging Face AI Agents Course](https://github.com/huggingface/agents-course) — going from beginner to building AI agents.

## What this covers

- **Agent fundamentals**: Tools, Thoughts, Actions, Observations, LLMs, messages, and chat templates.
- **Frameworks**: building agents with [smolagents](https://huggingface.co/docs/smolagents/en/index), [LlamaIndex](https://www.llamaindex.ai/), and [LangGraph](https://langchain-ai.github.io/langgraph/).
- **MCP**: how MCP tools work and the MCP protocol in depth.
- **Multi-agent systems**: using LangGraph and LangSmith.
- **Use cases & final assignment**: real-world agents and a benchmark challenge.

## Prerequisites

- Basic Python
- Basic understanding of LLMs

## Tools

- A computer with an internet connection
- A free [Hugging Face account](https://hf.co/join)
